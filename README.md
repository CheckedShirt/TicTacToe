# AI-TicTacToe
井字棋小游戏<br>
由C++编写的井字棋小游戏<br>
分人人对战与人机对战两种模式<br>
人机对战中的AI有三种智能等级<br>
    1级AI纯随机下子<br>
    2级AI会堵子<br>
    3级AI不会输<br>
